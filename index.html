Tentang Dashboard Ini

Smart Environment Dashboard dikembangkan untuk memberikan pemantauan suhu lingkungan secara real-time menggunakan sensor IoT. Data dikirim secara kontinu dan divisualisasikan untuk membantu pengguna memahami pola perubahan suhu di sekitarnya.

  • Visualisasi Data Interaktif
  • Berbasis IoT
